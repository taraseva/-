Здесь представленны данные и модули, которые объединяют оптимизацию социально-экономических показателей для роста миграционной привлекательности с оценкой потенциала социальных конфликтов на основе статистического сходства территорий. 
Данный метод позволяет формировать управленческие рекомендации и одновременно контролировать риск социальной нестабильности.

data — в этом каталоге содержатся все необходимые наборы данных
Forecasting model.ipynb - модель прогнозирования на основе случайного леса, которая предсказывает потенциал (общий балл по методу ранжирования) на следующий год
Обработка данных + модель + веса.ipynb - модель прогнозирования на основе случайного леса, которая предсказывает миграционное сальдо трудящихся
Формирование_данных_(норм_риск).ipynb – мэппинг данных по риску и миграционному сальдо
метод оптимизации.ipynb – метод оптимизации соц обеспечения населения
